Mini Trello App

A modern task management web application built using React.
This app helps users organize tasks with drag-and-drop functionality across different stages.

Features:

1) Add, Edit, Delete Tasks
2) Drag and Drop between columns (To Do, In Progress, Done)
3) Real-time Search & Filtering
4) Dark Mode Toggle
5) Data persistence using Local Storage
6) Fully Responsive (Mobile, Tablet, Desktop)

Technical Skills Used:

1) React.js
2) JavaScript (ES6)
3) Bootstrap 5
4) HTML5 & CSS3

Setup Instructions:

Follow these steps to run the project locally:

step-1) Clone the repository: git clone https://github.com/tejareddy121121/Mini-Trello-App.git

step-2) stepNavigate to project folder: cd Mini-Trello-App/task-flow

step-3) Install dependencies: npm install

step-4) Start the development server: npm run dev

step-5) Open in browser: http://localhost:5173
